# Сервис регистрации

## Для чего?

Сервис для регистрации новых пользователей

---

## API

Api предоставляет один _endpoint:_   
```http request
POST /api/v1/registration
```

Тело запроса:
```json
{
    "username": "qwerty",
    "password": "123"
}
```

Если _логин_ свободен, произойдет регистрация и автоматическая авторизация
